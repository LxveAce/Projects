# Security Policy

## Supported Versions

Only the latest commit on the default branch is supported.

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report security issues by emailing **lxveace@proton.me** directly. Include a description of the issue, steps to reproduce, and potential impact. You can expect an acknowledgment within 72 hours.

This repository contains documentation, guides, and reference material for personal security-research and hardware projects. It does not ship production software or libraries, so most security reports will relate to misleading/harmful guidance rather than exploitable code. All reports are taken seriously regardless.
